Project overview :- E-Commerce Website

Create E-commerce based platform with dynamic product listing, search, cart management, and order history features.

1. User-friendly interface with a responsive layout for all devices.
2. Users can add products, edit quantities, and remove items.
3. Displays a list of previous orders made by the user.
4. Includes an "Add to Cart" button for each product.
5. Cart updates automatically with recalculated totals and quantities.

Technologies Used: HTML , CSS & JAVASCRIPT
